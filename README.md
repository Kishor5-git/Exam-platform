# 🌌 ExamPro Universal: The Authoritative Assessment Frontier

**ExamPro Universal** is a premium, high-fidelity Online Examination & Learning Management Ecosystem. It transforms institutional assessment paradigms into a sleek, high-availability SaaS experience with integrated code execution and advanced performance analytics.

---

## 🏗️ Project Architecture
This ecosystem is architected as a professional monorepo, ensuring absolute synchronicity between the frontend and backend sectors.

- **`frontend/`**: Next.js (Turbo Manifold) application utilizing the "Aura" Glassmorphism design system.
- **`backend/`**: Node.js & Express API sector with high-intensity evaluation manifolds and Turso/LibSQL persistence.

---

## 🚀 Technical Manifest

### Prerequisites
- **Node.js**: v18+ Protocol
- **npm**: v9+ Coordination
- **Turso Database**: Authoritative Cloud/Local Registry

### Initialization Protocol

#### 1. Backend Sector Manifestation
```bash
cd backend
npm install
# Configure .env with TURSO_DATABASE_URL and TURSO_AUTH_TOKEN
npm run dev
```

#### 2. Frontend Sector Manifestation
```bash
cd frontend
npm install
npm run dev
```
